## 학습 방법론

실험과 검증을 통한 깊이 있는 이해를 추구합니다.

### 실험 기반 학습 사이클
1. 구체적인 질문 정의
2. 가설 세우기 (최소 2개 이상의 대안)
3. 실험 설계 (변수를 하나씩만 변경)
4. 결과 관찰 및 기록
5. 분석 및 새로운 질문 도출

### 핵심 원칙
- "비동기", "나중에" 같은 모호한 용어 대신 정확한 메커니즘 이해
- "무엇이 무엇을 블로킹"처럼 주체-대상을 명확히
- 숨겨진 가정 찾아내기
- 실제 환경과 유사한 실험 설계
