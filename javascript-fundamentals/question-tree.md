## 레벨 2: 다운로드 vs 실행 분리
**질문**: "나중에"는 무엇이 나중인가?

### 가설 2-A: 다운로드가 나중
- async/defer는 파일 다운로드 시점을 변경
- 실행은 다운로드 직후

### 가설 2-B: 실행이 나중
- 다운로드는 즉시 시작
- 실행만 지연

### 가설 2-C: 둘 다
- 다운로드와 실행 모두 영향받음

**검증 방법**: 실험 0.5 (Network 탭 분석)
