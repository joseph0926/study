# 레벨 0: 코드가 브라우저에 도달하는 과정

## 최상위 질문
**"내가 작성한 .js 파일이 어떻게 브라우저에서 실행되는가?"**

## 단계별 분해

### 1단계: 코드가 "어디"에 있는가?
- [ ] 내 로컬 컴퓨터의 파일 시스템에 있다
- [ ] 서버의 어딘가에 있다
- [ ] CDN에 있다

**질문**: 브라우저는 어떻게 이 파일의 "위치"를 알아낼까?

### 2단계: 브라우저가 파일을 "가져오는" 과정
- [ ] 브라우저는 어떻게 파일을 요청하는가?
- [ ] 파일은 어떤 형태로 전송되는가? (텍스트? 바이너리?)
- [ ] 전송 과정에서 압축/변환이 일어나는가?

### 3단계: 브라우저가 파일을 "받았다"는 것의 의미
- [ ] 브라우저의 "어디"에 저장되는가? (메모리? 디스크?)
- [ ] 파일 내용이 변경되는가? (인코딩 변환 등)

### 4단계: "실행 준비"란?
- [ ] 받은 텍스트를 브라우저가 "이해"한다는 것은?
- [ ] 언제부터 실행 가능한 상태가 되는가?
