실험 0.5: 언제 다운로드하고, 언제 실행하는가?

Network 탭에서 확인할 것:

  1. 세 파일의 다운로드 시작 시간이 같은가 다른가?
  2. 세 파일의 다운로드 완료 시간은?
  3. 다운로드 완료와 실행(콘솔 로그) 사이의 시간 차이는?

예측:

당신의 가설을 세워보세요:

  • async는 "다운로드를 비동기로" 하는가? "실행을 비동기로" 하는가? 둘 다?
  • defer는 "다운로드를 지연"하는가? "실행을 지연"하는가?

힌트:

Network 탭의 Waterfall을 자세히 보면:

  • 요청이 언제 시작되는지
  • 다운로드가 언제 완료되는지
